Freecodecamp URL Shortener Microservice

Takes a url as a parameter and returns a shortened url that routes to given url

  • If I pass paramenter "/new/[url]", browser will receive a shorted url in the JSON response to the GET request
  • The new URL will be of the form "[this app]/####" where # is a 4-digit number eg. 9392
  • If I enter the new URL I will be redirected to the URL originally associated with the shortened URL

Example of shortened URL Creation

https://linen-pound.glitch.me/new/https://www.google.com
https://linen-pound.glitch.me/http://foo.com:80

Example of JSON output in response to creation request

{ "original_url":"http://foo.com:80", "short_url":"https://linen-pound.glitch.me/8170" }

Example of Usage of shortened URL:

https://linen-pound.glitch.me/2871

In this example, shortened URL will redirect to:

https://www.google.com/
